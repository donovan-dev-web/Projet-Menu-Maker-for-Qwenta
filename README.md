Ce projet a été réalisé dans le cadre de ma formation en développement web.

Je suis intervenu en tant que **développeur front-end**, avec une responsabilité élargie incluant :

---

## Projet

Le client **Qwenta**, spécialisé dans l’impression de supports professionnels (restauration, immobilier, BTP), souhaite diversifier ses activités.

Dans ce contexte, il souhaite développer une application web appelée **Menu Maker**.

### Problématique

Permettre aux restaurateurs de créer, personnaliser et diffuser leurs menus **facilement et rapidement**, sans compétences techniques.

### Solution proposée

Une interface web intuitive permettant aux utilisateurs de :

* créer leurs menus en quelques clics
* personnaliser leur identité visuelle
* exporter et diffuser leurs menus
