# Projet-Menu-Maker-for-Qwenta
Votre client Qwenta souhaite faire développer un “Menu Maker” : un site permettant aux restaurateurs d’afficher et de mettre en page leurs menus facilement, en quelques clics.             Soufiane, le Product Owner du projet est en congé maladie. Il vous a chargé, exceptionnellement, de préparer et planifier ce projet.
