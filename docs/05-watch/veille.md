#### React Documentation
- **Type** : Documentation officielle

- **Pertinence** : Référence pour l’architecture SPA, composants, hooks et context API utilisés dans le projet.

#### React-PDF GitHub
- **Type** : Repo & Docs  
- **Lien** : [React-PDF](https://github.com/diegomura/react-pdf)  
- **Pertinence** : Pour l’export PDF côté client et la prévisualisation fidèle du menu.

#### MongoDB Manual
- **Type** : Documentation officielle  
- **Lien** : [MongoDB](https://www.mongodb.com/docs/)  
- **Pertinence** : Guide pour structurer les collections menus et utilisateurs, compatible JSON.

#### Node.js + Express Guide
- **Type** : Blog / Tutoriel  
- **Lien** : [Express](https://expressjs.com/)  
- **Pertinence** : Bonnes pratiques pour construire l’API REST sécurisée et gérer les erreurs.

- **Type** : Documentation API  
- **Lien** : [Deliveroo API](https://developer.deliveroo.com/)  
- **Pertinence** : Référence pour l’export automatique des menus vers la plateforme Deliveroo.

#### Instagram Graph API
- **Type** : Documentation API  
- **Lien** : [Instagram Graph API](https://developers.facebook.com/docs/instagram-api/)  
- **Pertinence** : Publication automatique des images de menus sur Instagram.

#### React State Management
- **Type** : Blog  
- **Lien** : [Medium Article](https://medium.com/@react_state)  
- **Pertinence** : Comparatif Context / Redux, justifiant le choix de Context pour ce projet.

#### FormData + Upload Files
- **Type** : Tutoriel  
- **Lien** : [MDN FormData](https://developer.mozilla.org/en-US/docs/Web/API/FormData)  
- **Pertinence** : Pour l’upload sécurisé des images (logos et plats) vers le backend / S3.
